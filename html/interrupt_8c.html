◆ vlc_interrupt_finish()

static int vlc_interrupt_finish ( vlc_interrupt_t * ctx)
static

Cleans up after an interruptible wait: waits for any pending invocations of the callback previously registed with vlc_interrupt_prepare(), and rechecks for any pending interruption.

Warning
As this function waits for ongoing callback invocation to complete, the caller must not hold any resource necessary for the callback to run. Otherwise a deadlock may occur.
Returns
EINTR if an interruption occurred, zero otherwise
